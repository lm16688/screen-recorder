# RecStudio Pro - 专业屏幕录制应用

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://[你的GitHub用户名].github.io/[仓库名])

一个功能强大的屏幕录制应用，支持摄像头叠加、白板工具和高质量录制。

## ✨ 功能特点

- 📹 **多种录制模式**：整个屏幕、窗口、自定义区域、摄像头
- 🎥 **摄像头叠加**：支持圆形/方形摄像头画面，可拖动位置
- 🎨 **白板工具**：画笔、线条、箭头、矩形、圆形、文本等
- 🎵 **音频选项**：系统音频、麦克风录制
- 📊 **多种质量**：720p、1080p、1440p可选
- 🎯 **实时预览**：录制过程中实时显示
